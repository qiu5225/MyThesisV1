# Q-learning 算法详解

Q-learning 是一种基于**价值迭代（Value-iteration）**的强化学习算法，属于**无模型（Model-free）**强化学习。它不依赖于环境的状态转移模型，而是通过与环境的交互来学习最优动作策略。

### 1. 核心概念

Q-learning 的核心思想是学习每个“状态-动作”对的价值函数 $Q(s,a)$，该函数用于评估在状态 $s$ 下执行动作 $a$ 后预期的总回报。其主要目标是通过更新 Q 值来逼近最优策略。

在 Q-learning 中，智能体（Agent）不断与环境交互，更新状态动作值 $Q(s,a)$，并最终收敛到最优解。其具体的更新公式如下：

$$Q(s_t,a_t)\leftarrow Q(s_t,a_t)+\alpha\left[r_t+\gamma\max_{a^{\prime}}Q(s_{t+1},a^{\prime})-Q(s_t,a_t)\right]$$

其中：

- $Q(s_t,a_t)$：表示在状态 $s_t$ 下采取动作 $a_t$ 的当前估计价值。
- $\alpha$：**学习率（Learning rate）**，控制更新的步长。
- $\gamma$：**折扣因子（Discount factor）**，代表未来回报的现值。

通过持续更新 Q 值，Q-learning 可以确定一个最优动作策略 $\pi(s)$，即在每个状态下应采取的最佳动作。

### 2. 马尔可夫决策过程 (MDP)

图 2 展示了 Q-learning 的过程：智能体通过与环境的持续交互来学习最优策略，从而实现累计回报最大化。图中描述的 MDP 主要由以下五个关键要素组成：

1. $S$：**状态集合**。
2. $A$：**动作集合**。
3. $\gamma$：**折扣因子**。
4. $r(s,a)$：**奖励函数**，奖励取决于状态 $s$ 和动作 $a$（若仅取决于状态 $s$，则退化为 $r(s)$）。
5. $P(s’|s,a)$：**状态转移函数**，表示在状态 $s$ 下执行动作 $a$ 后到达状态 $s'$ 的概率。

$$P(S_{t+1}|S_t)=P(S_{t+1}|S_1,...,S_t)$$

### 3. 实例说明：自动驾驶

想象一下驾驶汽车的场景：

- **环境 ($S_t$)**：周围的行人、树木和其他车辆构成了当前状态。
- **动作 ($A_t$)**：智能体在观察环境状态后选择动作，例如踩刹车减速。
- **反馈与更新**：执行动作后，环境发生变化并产生新状态 $S_{t+1}$，同时提供奖励 $R_{t+1}$。

奖励机制的作用是不断优化智能体的行为：

- 如果执行了**错误动作**，则给予负奖励。
- 如果执行了**正确动作**，则给予正奖励。

通过这种持续的交互，智能体不断接受训练，最终获得更优的性能表现。
