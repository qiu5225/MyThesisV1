## 3. 哈里斯鹰优化算法 (Harris Hawks Optimization, HHO)

哈里斯鹰优化算法是由 Heidari 等人于 2019 年提出的一种新型群智能寻优算法。该算法模拟了哈里斯鹰在捕食过程中的合作搜索、包围、驱赶及攻击行为。

### 3.1 勘探阶段 (Exploration Phase)

在 HHO 中，哈里斯鹰在搜索空间内随机寻找猎物。根据随机概率 $q$，算法采用以下两种等概率策略更新鹰的位置：

$$X(t+1) = \begin{cases} X_{rand}(t) - r_1 |X_{rand}(t) - 2r_2 X(t)|, & q \geq 0.5 \\ (X_{rabbit}(t) - X_m(t)) - r_3 (LB + r_4(UB - LB)), & q < 0.5 \end{cases}$$

其中，$X(t+1)$ 为第 $t+1$ 次迭代时鹰的位置，$X_{rabbit}(t)$ 为当前猎物（最优解）的位置，$r_1, r_2, r_3, r_4$ 和 $q$ 是 $(0, 1)$ 范围内的随机数。$LB$ 和 $UB$ 分别表示搜索空间的上下界。

### 3.2 从勘探到开发的过渡 (Transition Phase)

HHO 算法通过猎物的**逃逸能量 ($E$)** 实现从全局勘探到局部开发的动态转换。逃逸能量的数学模型定义为：

$$E = 2E_0 \left( 1 - \frac{t}{T} \right)$$

其中，$E_0$ 是介于 $(-1, 1)$ 之间的初始能量因子。当 $|E| \geq 1$ 时，算法执行勘探阶段；当 $|E| < 1$ 时，算法进入局部开发阶段。

### 3.3 开发阶段 (Exploitation Phase)

根据猎物的逃逸行为和鹰的包围策略，开发阶段分为以下四种模式：

1. **软包围 (Soft Besiege):** 当 $r \geq 0.5$ 且 $|E| \geq 0.5$ 时。此时猎物能量较充沛，试图通过跳跃逃脱，鹰采用缓慢包围策略。

2. **硬包围 (Hard Besiege):** 当 $r \geq 0.5$ 且 $|E| < 0.5$ 时。猎物极度疲劳，鹰直接进行猛烈突袭。

3. **渐进式快速俯冲软包围:** 当 $r < 0.5$ 且 $|E| \geq 0.5$ 时。引入莱维飞行 (Lévy flight, $LF$) 模拟猎物的不规则逃扑路径：

   $$Y = X_{rabbit}(t) - E |J X_{rabbit}(t) - X(t)|$$

   $$Z = Y + S \times LF(D)$$

4. **渐进式快速俯冲硬包围:** 当 $r < 0.5$ 且 $|E| < 0.5$ 时。在猎物能量极低时，鹰群通过减小与猎物平均位置的距离来实施攻击。

### 3.4 HHO 算法伪代码

Markdown

```
Algorithm: Harris Hawks Optimization (HHO)
Inputs: Population size N, Max iterations T
Outputs: The best location of rabbit (X_rabbit)

Initialize the random population X_i (i = 1, 2, ..., N)
While (t < T) do
    Calculate the fitness of each hawk
    Set X_rabbit as the best location
    For each hawk (X_i) do
        Update the initial energy E0 and jump strength J
        Update the escape energy E using Eq. (2)
        If (|E| >= 1) then
            Update location using Exploration Phase (Eq. 1)
        Else if (|E| < 1) then
            If (r >= 0.5 and |E| >= 0.5) then Soft Besiege
            Else if (r >= 0.5 and |E| < 0.5) then Hard Besiege
            Else if (r < 0.5 and |E| >= 0.5) then Soft Besiege with progressive rapid dives
            Else if (r < 0.5 and |E| < 0.5) then Hard Besiege with progressive rapid dives
        End if
    End for
    t = t + 1
End while
Return X_rabbit
```
